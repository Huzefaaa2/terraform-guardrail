# Examples

The [examples/](https://github.com/Huzefaaa2/terraform-guardrail/tree/main/examples) folder provides
hands-on CI simulations and sample outputs.

## What's included

- [examples/fixtures/](https://github.com/Huzefaaa2/terraform-guardrail/tree/main/examples/fixtures)
  contains good and bad Terraform inputs (`.tf`, `.tfvars`, `.hcl`, `.tfstate`).
- [examples/gitlab-ci/](https://github.com/Huzefaaa2/terraform-guardrail/tree/main/examples/gitlab-ci)
  shows a GitLab CI template example and sample outputs.
- [examples/github-actions/](https://github.com/Huzefaaa2/terraform-guardrail/tree/main/examples/github-actions)
  shows a GitHub Actions workflow example and sample outputs.

Each folder includes a `README.md` with steps and expected output formats (JSON, SARIF, JUnit,
CSV).
