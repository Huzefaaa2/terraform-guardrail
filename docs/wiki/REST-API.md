# REST API

Terraform Guardrail MCP (TerraGuard) exposes a REST API for CI/CD usage and UI integrations.

## Purpose

- Validate Terraform configs and state outside of Terraform.
- Provide consistent guardrail results for pipelines and dashboards.
- Power the Streamlit UI and registry workflows.

## Status

Delivered as part of the Dockerized MCP + REST API milestone.
